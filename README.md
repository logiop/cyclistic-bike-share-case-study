# Cyclistic Bike-Share Case Study

## Google Data Analytics Professional Certificate - Capstone Project

### Overview
This case study analyzes how annual members and casual riders use Cyclistic bikes differently, with the goal of designing marketing strategies to convert casual riders into annual members.

### Business Task
Understand behavioral differences between annual members and casual riders to inform targeted marketing strategies for membership conversion.

### Data Sources
- **Divvy_Trips_2019_Q1.csv** - 365,069 rides from Q1 2019
- **Divvy_Trips_2020_Q1.csv** - 426,887 rides from Q1 2020
- **Total**: 791,264 rides analyzed

*Note: CSV data files are not included in this repository due to size. Download from [Divvy Trip Data](https://divvy-tripdata.s3.amazonaws.com/index.html)*

### Key Findings

| Metric | Members | Casual Riders |
|--------|---------|---------------|
| Ride Share | 91% | 9% |
| Avg Duration | 11.4 min | 36.5 min |
| Peak Days | Tue-Thu | Sunday |
| Peak Hours | 8AM, 5PM | 2-4PM |

**Insight**: Casual riders ride **220% longer** than members, indicating leisure-oriented usage vs commute-focused membership usage.

### Top 3 Recommendations

1. **Weekend Membership Promos** - Target casual riders with special weekend offers showing cost savings
2. **Corporate Commuter Program** - Partner with businesses to offer membership as employee benefit
3. **Digital Cost Calculator** - In-app tool showing personalized savings based on usage patterns
